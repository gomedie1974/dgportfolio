Tecnologías que uso y recomiendo

Estas son algunas de las tecnologías que utilizo en mis proyectos. Cada una aporta diferentes ventajas y funcionalidades para crear experiencias web modernas, rápidas y eficientes.

HTML5

Lenguaje de marcado esencial para estructurar contenido web de forma semántica y accesible.

CSS3

Permite diseñar sitios atractivos y responsivos, aplicando estilos, animaciones y layouts modernos.

JavaScript

Lenguaje de programación para hacer páginas web interactivas y dinámicas, desde animaciones hasta apps completas.

React

Framework para construir interfaces modernas, rápidas y escalables usando componentes reutilizables.

Bootstrap

Framework CSS que agiliza el diseño responsive con componentes predefinidos y grid flexible.

Python

Lenguaje versátil para desarrollo web, automatización, data science y machine learning.

PHP

Lenguaje de backend ampliamente usado para construir sitios web dinámicos y gestionar bases de datos.

SQL

Lenguaje para manejar bases de datos relacionales, consultas y optimización de datos.

WordPress

Sistema de gestión de contenidos (CMS) para crear sitios web y blogs de forma rápida y flexible.

GitHub

Plataforma para control de versiones y colaboración en proyectos de desarrollo.

Firebase

Suite de herramientas para desarrollo de apps web y móviles con backend en tiempo real.

NotebookLM

Herramienta de IA que ayuda a organizar, consultar y procesar información de manera inteligente.
